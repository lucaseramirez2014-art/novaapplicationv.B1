# Nova - AI-Powered Productivity Ecosystem

Nova is an AI-powered productivity ecosystem built to help people manage the parts of life that usually feel scattered — money, goals, and eating well — in one connected system instead of three disconnected apps.

## Overview

Each Nova app focuses on a different part of daily life, but they all share the same core idea: AI that actively helps you adjust and stay on track, not just track and report.

### NovaFinance
Automatically detects recurring subscriptions, warns you before bills are due, and learns your spending patterns over time, so managing money feels less like manual bookkeeping and more like having a second set of eyes.

### NovaGoals
Breaks big goals into clear, manageable steps and re-plans with you the moment you get stuck, with forgiving streaks and guilt-free archiving instead of the all-or-nothing pressure most productivity apps create.

### NovaFit
Builds meal plans that are actually tasty, genuinely healthy, and realistic on a budget, with dietary restrictions built in so vegetarian, allergies, or other needs are respected automatically instead of requiring manual filtering. Every plan comes with a ready-to-go shopping list.

## The Nova Philosophy

People don't fail their goals — financial, personal, or nutritional — because they lack ambition. They fail because rigid systems don't adapt when life gets messy. Nova is designed to adapt with you.

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No build tools or dependencies required

### Installation

1. Clone or download this repository
2. Navigate to the project directory
3. Open `index.html` in your web browser

That's it! The application runs entirely in the browser with no server or build process required.

### Quick Start

Simply open `index.html` in your browser to start using Nova. The application includes:

- **Hub**: Central dashboard connecting all three apps
- **NovaFinance**: Financial management with subscription tracking
- **NovaGoals**: Goal setting with adaptive planning
- **NovaFit**: Meal planning with dietary restrictions

## Features

### NovaFinance
- **Subscription Detection**: Automatically identifies recurring charges
- **Bill Warnings**: Get alerted before bills are due
- **Spending Patterns**: AI learns your habits over time
- **Spending Overview**: Visual breakdown of expenses
- **Cross-App Sync**: Connects with NovaFit for budget-friendly meal planning

### NovaGoals
- **Goal Breakdown**: Automatically breaks big goals into steps
- **Adaptive Replanning**: AI suggests adjustments when you're stuck
- **Forgiving Streaks**: Maintain motivation without guilt
- **Guilt-Free Archiving**: Pause goals without feeling like a failure
- **Progress Tracking**: Visual progress bars and milestone tracking

### NovaFit
- **Smart Meal Planning**: Tasty, healthy meals on a budget
- **Dietary Restrictions**: Built-in support for vegetarian, gluten-free, allergies, and more
- **Shopping Lists**: Ready-to-go lists organized by category
- **Cost Optimization**: AI adjusts plans based on your budget
- **Cross-App Sync**: Integrates with NovaFinance for budget-aware planning

## Navigation

Use the navigation bar at the top to switch between apps:
- **Hub**: Overview and quick access to all apps
- **Finance**: NovaFinance dashboard
- **Goals**: NovaGoals dashboard
- **Fit**: NovaFit dashboard

Keyboard shortcuts:
- Press `1` for Hub
- Press `2` for Finance
- Press `3` for Goals
- Press `4` for Fit
